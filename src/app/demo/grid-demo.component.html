การใช้งาน

Classes ที่สามารถใช้ได้:

  • .pk-row - สร้าง grid container
  • .pk-col-(1-12) - กำหนดขนาดคอลัมน์สำหรับทุกขนาดหน้าจอ
  • .pk-col-sm-(1-12) - กำหนดขนาดคอลัมน์สำหรับหน้าจอขนาดเล็ก (640px+)
  • .pk-col-md-(1-12) - กำหนดขนาดคอลัมน์สำหรับหน้าจอขนาดกลาง (768px+)
  • .pk-col-lg-(1-12) - กำหนดขนาดคอลัมน์สำหรับหน้าจอขนาดใหญ่ (1024px+)
  • .pk-col-xl-(1-12) - กำหนดขนาดคอลัมน์สำหรับหน้าจอขนาดใหญ่มาก (1280px+)
  • .pk-row-gap-(0,1,2,3,4,5,6,8) - กำหนดระยะห่างระหว่างคอลัมน์
