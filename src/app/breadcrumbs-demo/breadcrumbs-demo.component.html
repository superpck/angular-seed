Usage Information

CSS Classes Available:

  • .pk-breadcrumbs - Base breadcrumb container
  • .pk-breadcrumbs-background - Background styling variant
  • .pk-breadcrumbs-colored - Colored accent variant
  • .pk-breadcrumbs-large - Large size variant
  • .pk-breadcrumb-list - Breadcrumb items list
  • .pk-breadcrumb-item - Individual breadcrumb item
  • .pk-breadcrumb-link - Clickable breadcrumb links
  • .pk-breadcrumb-text - Non-clickable breadcrumb text
  • .pk-breadcrumb-active - Active/current page indicator
  • .pk-breadcrumb-icon - Icon container within breadcrumbs

Accessibility Features:

  • Semantic HTML structure with nav and ol elements
  • ARIA labels for screen readers
  • Keyboard navigation support
  • Focus management and hover states
  • Clear visual separation between links and current page
